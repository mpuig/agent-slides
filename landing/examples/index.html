How these were built

Each example follows the same workflow:

  1. Download a template from Microsoft's free template gallery
  2. Extract the template with /slides-extract to produce layout contracts and a design profile
  3. Build the deck with /slides-full from a single natural-language prompt

The agent handles planning, layout selection, ops generation, rendering, and QA automatically. No manual editing after generation.

Source templates

All three templates are freely available from Microsoft:

  • Example 1: Nature-themed presentation — 52 layouts, earthy color palette
  • Example 2: Geometric presentation — clean lines, professional layout
  • Example 3: Public speaking presentation — 12 layouts, photo-driven design
