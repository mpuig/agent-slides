Example — 5 content slides

Quarterly revenue review.

A compact board-level update covering ARR growth, segment bookings, regional quota attainment, NRR trends, and Q4 investment priorities.

The prompt

Create a 5-slide quarterly revenue review for the Board of Directors using /slides-full.

Audience: Board of Directors, CEO, CFO.
Tone: Concise executive update (data-forward, KPI-driven).
Scope: ARR growth ($127M, +34% YoY), segment bookings breakdown,
regional quota attainment, net revenue retention trends,
Q4 investment priorities.

Use a bar chart for segment bookings, a line chart for NRR trend,
and a data table for regional performance. Include big-number callouts
for headline KPIs and source lines on every data slide.

Template

Source: Microsoft free template gallery — a nature-themed 16:9 template with 52 layouts, earthy tones (accent colors: #A57361, #4E6C67, #CD9979), and photo-driven cover slides.

Reproduce it

# 1. Download the template from Microsoft

# 2. Extract
/slides-extract example1.pptx

# 3. Build (paste the prompt above)
/slides-full "5-slide quarterly revenue review ..."

# Or render from the JSON directly
slides render \
  --slides-json @slides.json \
  --profile design-profile.json \
  --output output.pptx
